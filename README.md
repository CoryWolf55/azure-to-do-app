# Task Tracker (React Frontend)

## Overview

This is a **basic Task Tracker app** built with React.

Currently, only the **frontend GUI** is set up to add and view tasks locally.

The goal of this project is to learn and integrate:

- **.NET Web API** to manage tasks
- **Azure Functions** for automated reminders
- **Azure App Service** and **Azure Static Web Apps** for deployment

## Current Status

- React frontend is functional locally
- Users can add tasks and see them listed in the UI
- No backend integration yet

## Planned Features

- Connect frontend to a **.NET Web API** for task storage
- Implement **Azure Function** to check tasks on a schedule
- Deploy React frontend and API to **Azure free-tier services**
- Optional enhancements: persistent storage, notifications, styling

## Getting Started

### Prerequisites

- **Node.js + npm**
- React development environment
